核心组件

01

Tokenizer

将文本字符串转换为整数序列（tokens）。基于唯一字符构建词汇表，支持特殊的序列开始 (BOS) token。

02

自动微分 Autograd

实现完整的反向传播算法。通过计算图进行链式法则计算，Value 类跟踪前向和后向过程。

03

Transformer

单层 Transformer 模型：多头注意力机制、前馈神经网络 (MLP)、RMSNorm 层正则化、ReLU 激活函数。

04

训练 & 推理

交叉熵损失函数用于下一个 token 预测。简单 SGD 优化。训练完成后生成新的序列。

模型架构

默认配置

python 
# Transformer 超参数
n_layer    = 1     # Transformer 层数
n_embd     = 16    # 嵌入维度
block_size = 16    # 最大上下文长度
n_head     = 4     # 注意力头数

数据流

输入文本
↓
Tokenizer 编码
↓
Token 嵌入 + 位置编码
↓
Transformer Block
多头注意力 → MLP → RMSNorm
↓
Logits 输出
↓
Softmax → 生成
